## Sections

1. **Home** - Hero section with company introduction and statistics
2. **About** - Company story and values
3. **Services** - Cleaning service offerings
4. **Gallery** - Before/after photos and work examples
5. **Contact** - Contact form and company information

## Customization

- Colors: Edit CSS variables in `css/style.css` (lines 8-25)
- Content: Update text in `index.html`
- Images: Replace images in `/images` directory
- Contact info: Update contact details in the Contact section

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
